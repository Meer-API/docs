### Step 1: Sign up for an API Key

Request a key at [the Meer API official website](https://www.meerapi.com/) - this process will soon be automated.

### Step 2: Start querying

Use your token to query our HTTPS API

<RequestExample>

```filename
curl --request GET \
  --url https://api.meerapi.com/check_phone_number \
  --header 'Authorization: Bearer <token>'
```

</RequestExample>
